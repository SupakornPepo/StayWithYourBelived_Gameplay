# StayWithYourBelief

##### เป็นเกม First Person แนว Horror, Mystery โดยมี Theme ของเกมคือ ล้านนา จึงได้ใช้ประเพณีทางภาคเหนือต่าง ๆ นำมาปรับใช้ภายในเกม เช่น ประเพณีทางหลัวหิงไฟพระเจ้า ประเพณียี่เป็ง เป็นต้น และได้สอดแทรกปริศนาคำทายล้านนา เป็นคำถามที่ให้ผู้เล่นได้แก้ Puzzle ภายในเกม โดยเกมนี้ได้ใช้ Unity C# ในการพัฒนาขึ้นมา

## เกมนี้เป็นเกมทำระหว่างเรียนอยู่ชั้นปีที่ 3 มหาวิทยาลัยเชียงใหม่ อาจมีบัคและข้อผิดพลาดเล็กน้อย
